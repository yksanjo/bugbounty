### 3. Immunefi 💰 **HIGHEST PAYOUTS ($10M+)**
- **URL**: https://immunefi.com/signup/
- **Focus**: DeFi, Smart Contracts, Web3
- **Payout Range**: $1,000 - $10,000,000+
- **Requirements**:
  - Email + Wallet address
  - No KYC for small bounties
  - Smart contract knowledge helpful
- **Best For**: Critical smart contract bugs, frontend exploits on crypto sites
- **Registration Steps**:
  1. Go to https://immunefi.com/signup/
  2. Enter email + password
  3. Connect Discord (for community)
  4. Add wallet address (for payouts)

### 4. Code4rena 🏟️ **Competitive Audits ($50K-$500K)**
- **URL**: https://code4rena.com/
- **Focus**: Code audits, competitive contests
- **Payout Model**: Prize pool split among finders
- **Requirements**:
  - GitHub account (login via GitHub)
  - Smart contract reading ability
  - OR web2 security skills (many contests have web targets)
- **How It Works**:
  1. Contest opens (usually 5-7 days)
  2. You audit the code/target
  3. Submit findings
  4. Prize pool ($50K-$500K) split among all valid finders
- **Registration**:
  1. Go to https://code4rena.com/
  2. Click "Sign In" → GitHub OAuth
  3. Join Discord: https://discord.gg/code4rena
  4. Browse contests: https://code4rena.com/contests
- **Strategy for Beginners**:
  - Start with "Web" contests (not Solidity)
  - Look for auth, logic bugs, API issues
  - Even 1 valid bug = share of prize pool

### 5. Synack 🏛️ **INVITE-ONLY (Gov/Enterprise)**
- **URL**: https://www.synack.com/join-the-synack-community/
- **Status**: **Requires Application**
- **Payouts**: Highest per-bug ($500-$50K+)
- **Requirements**:
  - Background check (government clearance level)
  - Technical interview/vetting
  - LinkedIn profile with security experience
  - Resume/CV
- **Application Process**:
  1. Fill application: https://www.synack.com/join-the-synack-community/
  2. Submit LinkedIn + Resume
  3. Wait for review (2-4 weeks)
  4. Technical interview (if accepted)
  5. Background check
  6. Get invited to private programs
- **Why It's Worth It**:
  - Less competition (vetted hackers only)
  - Government contracts (DoD, DHS)
  - Consistent work
  - Higher payouts per bug
- **Strategy**: Apply NOW, wait for approval while hacking on other platforms
