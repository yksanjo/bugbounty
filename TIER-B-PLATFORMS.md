# 🥈 Tier B Platforms - Less Competition, Decent Payouts

## These are underrated but still paying platforms

### 1. Hackrate 🇩🇪 **EU Gov Programs**
- **URL**: https://hackrate.com/
- **Focus**: European government, mid-size companies
- **Payouts**: €100 - €10,000
- **Why It's Underrated**:
  - Most researchers don't speak German/Dutch
  - Government programs = stable budgets
  - Less competition than H1/Bugcrowd
- **Registration**:
  1. Go to https://hackrate.com/
  2. Click "Become a Hacker"
  3. Email verification
  4. Complete profile
  5. Browse programs

---

### 2. Bounty Factory 🏭 **Smaller Companies**
- **URL**: https://bountyfactory.io/
- **Focus**: Startups, mid-size companies
- **Payouts**: $50 - $5,000
- **Why It's Good**:
  - Less hardened targets
  - Startups = more bugs
  - Faster triage
- **Registration**:
  1. Go to https://bountyfactory.io/
  2. Sign up as researcher
  3. Email verification
  4. Browse programs

---

### 3. Zerocopter 🌐 **EU-Focused**
- **URL**: https://zerocopter.com/
- **Focus**: European companies, responsible disclosure
- **Payouts**: €50 - €20,000 (some paid, some VDP)
- **Why It's Good**:
  - Mix of paid + VDP programs
  - Good for building reputation
  - EU data protection focus
- **Registration**:
  1. Go to https://zerocopter.com/
  2. Sign up as security researcher
  3. Email verification
  4. Browse programs

---

### 4. SafeHats 🎓 **Educational + Paid**
- **URL**: https://www.safehats.com/
- **Focus**: Beginners, educational, some paid
- **Payouts**: $50 - $2,000
- **Why It's Good**:
  - Great for learning
  - Supportive community
  - Easier bugs for beginners
- **Registration**:
  1. Go to https://www.safehats.com/
  2. Sign up
  3. Email verification
  4. Browse programs

---

### 5. Open Bug Bounty 🆓 **Free/Reputation**
- **URL**: https://www.openbugbounty.org/
- **Focus**: XSS, CSRF (no monetary rewards)
- **Payouts**: Reputation points only
- **Why It's Useful**:
  - Build reputation score
  - Practice report writing
  - Good for HackerOne/Bugcrowd applications
- **Registration**:
  1. Go to https://www.openbugbounty.org/
  2. Register
  3. Submit first vulnerability
  4. Build reputation

## 🎯 Strategy for Tier B

### Week 1-2: Reputation Building
- Submit 2-3 bugs to Open Bug Bounty
- Build reputation score
- Learn report writing

### Week 3-4: First Paid Submissions
- Try Hackrate (EU gov programs)
- Submit to Bounty Factory (startups)
- Focus on XSS, IDOR, Open Redirect

### Month 2-3: Consistent Income
- Mix of Tier A + Tier B platforms
- Target: $500-$1,500/month
- Use Tier B for "easier wins" while building Tier A reputation

## 💡 Pro Tips for Tier B

1. **Hackrate**: Learn basic German/Dutch company names
2. **Bounty Factory**: Target newly launched programs (less scrutiny)
3. **Zerocopter**: Focus on Dutch/Belgian companies
4. **SafeHats**: Use for practice before submitting to paid platforms
5. **Open Bug Bounty**: Submit every valid XSS/CSRF you find
